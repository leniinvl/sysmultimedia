Introducción

El propósito del proyecto es crear una página Web como aporte final de la asignatura Sistemas Multimedia, la cual estará compuesta de teoría y aplicación de animaciones con el uso del objeto canvas con JavasScript en HTML, brindando un recurso multimedia para el buen aprovechamiento de los recursos. Es esencial mencionar que para el correcto funcionamiento del proyecto este debera estar alojado en un Servidor HTTP Apache como puede ser con Xammp o WampServer entre los mas conocidos.

Canvas (o lienzo) es un elemento incorporado en HTML5 que permite la generación de gráficos dinámicamente por medio del scripting. Entre otras cosas, permite la renderización interpretada dinámica de gráficos 2D y mapas de bits, asi como animaciones con estos gráficos. Se trata de un modelo de procedimiento de bajo nivel, que actualiza un mapa de bits y no tiene una gráfica de escena integrada. El objeto canvas permite contener diferentes objetos, incluyendo imágenes y videos, lo que permite aplicar varios procedimientos con el uso de JavaScript para transformarlos y animarlos.

Para crear un elemento CANVAS se aplica la siguiente estructura a nivel de HTML:


<canvas height="300px" width="300px" id="lienzo"> </canvas>

En Javascript, sus objetos de definen e inicializan:


var micanvas; 
var lienzo; 
function iniciar(){
		micanvas = document.getElementById("lienzo");
		lienzo = micanvas.getContext("2d");
}
window.addEventListener("load",iniciar,false);
